# Ethos: Moral Decision Logger (Backend API)

Internal RESTful API for the Ethos: Moral Decision Logger web application.

Built with Django REST Framework and PostgreSQL (Neon Cloud). Deployed on Render and used by a Vercel-hosted frontend.

---

## Stack

- Python Django REST Framework, ReactJS Vite TypeScript, TailwindCSS
- PostgreSQL (Neon Cloud)
- JWT Authentication (SimpleJWT)
- Render (Backend)
- Vercel (Frontend)
